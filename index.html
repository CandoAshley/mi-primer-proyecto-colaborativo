Tecnologías de la Información

La carrera de Tecnologías de la Información forma profesionales capaces de desarrollar soluciones tecnológicas, administrar sistemas informáticos, gestionar bases de datos y utilizar herramientas digitales para resolver problemas en empresas e instituciones.

¿Qué se aprende en la carrera?

En esta carrera se estudian temas como programación, desarrollo web, bases de datos, redes de computadoras y ciberseguridad. También se aprende a trabajar con herramientas tecnológicas para resolver problemas reales.

Habilidades que se desarrollan

  • Programación básica y avanzada
  • Diseño de páginas web
  • Administración de bases de datos
  • Trabajo en equipo
  • Uso de Git y GitHub
