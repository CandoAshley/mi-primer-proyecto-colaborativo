# Mi Primer Proyecto Colaborativo

## Objetivo del proyecto
Aprender a utilizar Git y GitHub para trabajar en proyectos colaborativos y conocer el control de versiones.

## Integrante
- Ashley Cando

## Herramientas colaborativas investigadas
- Git
- GitHub
- Commit
- Branch
- Merge

## Mejoras del proyecto
Este proyecto fue mejorado utilizando una rama llamada **mejoras**, donde se realizaron cambios sin afectar la versión principal del repositorio. Gracias a esto, se pudo trabajar de forma ordenada, segura y sin perder el control de las versiones del proyecto.

## Aprendizaje
Este proyecto permitió comprender la importancia del trabajo colaborativo en desarrollo de software, el uso de ramas para organizar cambios y los commits para registrar el progreso del proyecto.
